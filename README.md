# madoguchi デザインシステム

madoguchi が運営するサービスメディアのスライド・LP・UI制作に使用する共通デザインシステムです。

## リポジトリ構成

```
design-system/
├── README.md
└── 不用品回収の窓口/
    ├── CHANGELOG.md       # 変更履歴
    └── designsystem.md    # 最新版デザインシステム仕様書
```

今後、他メディアのデザインシステムも同階層に追加予定です。

```
design-system/
├── 不用品回収の窓口/
├── gaiheki+/
├── おそうじ合衆国/
├── みんなの買取/
└── 解体相談所/
```

## 運用ルール

- `designsystem.md` は常に**最新版**を反映する。
- 変更時は必ず `CHANGELOG.md` にバージョン・日付・変更概要を記載してからコミットする。
- コミットメッセージのフォーマット：`[メディア名] vX.X - 変更概要`
  - 例：`[不用品回収の窓口] v1.1 - フォント指定をNoto Sans JP + Interに更新`
- ブランチは切らず `main` に直接コミットする（小規模チーム運用）。
